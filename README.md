# 🧠 BrainBlast — Gamified Quiz Platform

**BrainBlast** is an interactive, gamified quiz platform that makes learning fun. Answer questions, earn XP, unlock badges, level up, and compete on the leaderboard!

---

## 🎮 Features

| Feature | Description |
|---------|-------------|
| ✅ **4 Categories** | Science, Math, History, Geography |
| ✅ **28 Questions** | 7 questions per category with 3 difficulty levels |
| ✅ **XP System** | Earn XP for correct answers (20/30/50 based on difficulty) |
| ✅ **Level System** | 100 XP per level; level up with visual celebration |
| ✅ **Badges** | 8 unlockable achievements (Streak, Level, XP, Perfect Quiz) |
| ✅ **Streak System** | 5-dot display; bonus XP at 3+ consecutive correct answers |
| ✅ **Timer** | 15-second countdown; bonus for fast answers |
| ✅ **Coins** | Earn coins for correct answers (1-5 based on difficulty) |
| ✅ **Leaderboard** | Compete with NPCs and track your rank |
| ✅ **Real-time Feedback** | Instant visual feedback (Correct/Wrong) with XP gain |
| ✅ **Particle Animations** | Dynamic background particles for visual appeal |
| ✅ **Toast Notifications** | Floating messages for streaks, badges, and level ups |
| ✅ **Level Up Modal** | Celebratory popup on each level up |

## 🚀 How to Play

1. **Start the Game**: Click "Start Playing" on the home screen
2. **Choose a Category**: Select Science, Math, History, or Geography
3. **Answer Questions**: Click the correct answer within 15 seconds
4. **Earn Rewards**: Gain XP, coins, and build your streak
5. **Unlock Badges**: Reach milestones to earn achievements
6. **Level Up**: Every 100 XP advances your level
7. **Check Leaderboard**: See how you rank against others
8. **Play Again**: Try different categories or improve your score
